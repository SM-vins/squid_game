GAME INSTRUCTIONS:

--> The user has 30 seconds to complete the glass bridge between the starting and finishing points. Initially, you can see five distinct blocks that function as glass.

--> In essence, users have five consecutive chances to choose the correct glass. If they choose the correct glass, they would score five points; if not, they would lose the game.

--> Additionally, the game indicates which mirror is accurate when the user chooses the hint. However, you would only receive 2 points and the timer would subtract 5 seconds for using the hint.
